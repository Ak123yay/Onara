# ADR-002 — Cloudflare Pages Over Vercel for User Sites

**Status**: Accepted | **Date**: 2026-05-14 | **Source**: raw/Onara_Business_Plan_v12.md

---

## Context

Onara hosts a generated website for every user. The solution must be free, scalable, and safe from surprise billing on traffic spikes or DDoS.

---

## Decision

Use Cloudflare Pages Direct Upload API for all user-generated websites. Each site gets a `{projectId}.pages.dev` subdomain.

---

## Rationale

1. **Free**: No commercial restrictions. Unlimited sites, unlimited bandwidth, 500 builds/month on free tier.
2. **No DDoS billing risk**: unlike Vercel, Cloudflare does not charge bandwidth overages. DDoS mitigation included.
3. **Direct Upload API**: programmatic deployment — FastAPI generates HTML and pushes via API with no Git repo connection needed.
4. **Global CDN**: fast delivery from edge for local business customers worldwide.
5. **Separation of concerns**: user sites on `*.pages.dev` are separate from `onara.tech` (Vercel) — user traffic can't affect app availability.

---

## Implementation

- FastAPI → Cloudflare Pages Direct Upload API at pipeline end
- Backup: GitHub commit to private `onara-sites/{projectId}/` repo
- Backup: Supabase storage bucket (for redeployment after cancellation/resubscription)
- Cancellation: real site → suspension placeholder page; redeployed from backup on resubscription
- Dashboard preview: served via protected `/api/preview/:project_id` — not the public URL. Free users always have preview.

---

## Consequences

- URL: `https://onara-site-{projectId[:20]}.pages.dev` by default
- Custom domain = $10 add-on
- URL changes on each redeployment cycle — documented in ToS

---

## Rejected Alternatives

| Alternative | Reason |
|-------------|--------|
| Vercel (user sites) | Bandwidth overage billing; commercial use prohibited on free tier |
| AWS S3 + CloudFront | Per-GB egress costs at scale; setup complexity |
| Netlify | Per-site limits on free plan; not designed for programmatic multi-site |
| GitHub Pages | Requires public repos or Pro for private pages |
